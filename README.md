> **Kubernetes Doctor** is an AI-powered assistant that helps users understand Kubernetes cluster issues, analyze logs, and suggest safe fixes in simple language.

---

## Overview

Kubernetes troubleshooting can be confusing and time-consuming, especially when logs, events, pod states, and node health are spread across different places.  
This project aims to simplify that process with an intelligent assistant that explains issues clearly and recommends safe next steps.

## Project Goal

The goal of this project is to build a smart Kubernetes troubleshooting assistant that can:
- understand cluster issues,
- summarize logs and events,
- identify likely root causes,
- and suggest safe remediation steps.
